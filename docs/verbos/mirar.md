# Mirar / Ver
Infinitivo: `shaf` / `yshuf`

| Castellano | Marroquí         |
|:-----------|:-----------------|
| Yo         | ana kan`shuf`    |
| Tu (m)     | nta kat`shuf`    |
| Tu (f)     | nti kat`shuf`y   |
| El         | huwa kay`shuf`   |
| Ella       | hiya kat`shuf`   |
| Nosotr@s   | Hna kan`shuf`u   |
| Ustedes    | ntuma kat`shuf`u |
| Ell@s      | huma kay`shuf`u  |
