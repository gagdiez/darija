# Leer
Infinitivo: `qra`

| Castellano | Marroquí         |
|:-----------|:-----------------|
| Yo         | ana kan`qra`    |
| Tu (m)     | nta kat`qra`    |
| Tu (f)     | nti kat`qra`y   |
| El         | huwa kay`qra`   |
| Ella       | hiya kat`qra`   |
| Nosotr@s   | Hna kan`qra`u   |
| Ustedes    | ntuma kat`qra`u |
| Ell@s      | huma kay`qra`u  |
