# Decir
Infinitivo: `gal` / `ygul`

| Castellano | Marroquí         |
|:-----------|:-----------------|
| Yo         | ana kan`gul`    |
| Tu (m)     | nta kat`gul`    |
| Tu (f)     | nti kat`gul`y   |
| El         | huwa kay`gul`   |
| Ella       | hiya kat`gul`   |
| Nosotr@s   | Hna kan`gul`u   |
| Ustedes    | ntuma kat`gul`u |
| Ell@s      | huma kay`gul`u  |
